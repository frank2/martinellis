1. Martinellis package¶

The root of the Martinellis package comes with just a helper function to assist the functionality of the underlying modules. It’s not necessarily related to CIDR functionality, but is very helpful for creating a memory-safe iterator over large ranges such as IPv6!

martinellis.xlongrange(start=None, stop=None)¶

Creates a range iterator similar to xrange, but for large numbers. Works pretty much like the range() function, in that it generates a range from 0 -> start non-inclusive if stop is not specified, or start -> stop non-inclusive if it is.
